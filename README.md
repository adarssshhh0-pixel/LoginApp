# Employee Management System

A full-stack Employee Management System built using **React.js, Node.js, Express.js, and PostgreSQL**. This application helps organizations manage employees, departments, roles, leave requests, skills, notifications, and reports through a centralized dashboard.

## Features

* User Authentication & Authorization
* Employee Management
* Department Management
* Role Management
* Leave Application & Approval
* Employee Skills Tracking
* Notifications System
* Reports & Analytics
* Protected Routes
* Profile Management

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios
* CSS

### Backend

* Node.js
* Express.js
* PostgreSQL
* JWT Authentication
* Multer
* Winston Logger

## Project Structure

```text
frontend/   -> React Application
backend/    -> Node.js & Express API
```

## Installation

### Clone the repository

```bash
git clone YOUR_REPOSITORY_URL
cd LoginApp
```

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file and configure your environment variables.

Start the backend:

```bash
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

The application will run on:

* Frontend: http://localhost:3000
* Backend: http://localhost:5000

## Environment Variables

Create a `.env` file inside the backend directory:

```env
PORT=5000
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
```
